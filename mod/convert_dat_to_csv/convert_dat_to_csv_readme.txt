The first row contains only the sampling rate in column 1 and zeros in other columns
The remaining columns have the stimulus information and data with each row representing one time point (with a time step of 1/sampling rate).
Column 1: Stimulus Code (numerical code for each stimulus, 0 for no stimulus)
Column 2: Simulus Type (1 for positive stimulus, 0 otherwise)
Column 3: Data for channel 1
Column 4: Data for channel 2
...
Column n+2: Data for channel n
